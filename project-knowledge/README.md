# project-knowledge/ — בסיס הידע הקבוע של הקורס

בסיס ידע שמגדיר את ה"DNA" של הקורס — הזהות היציבה שצריכה להישמר גם כשמוסיפים, מוחקים,
משכתבים או מעצבים מחדש תוכן. כל מפתח או עוזר AI עתידי צריך לקרוא כאן לפני שינוי.

## סדר קריאה
התחל תמיד מ-**COURSE_DNA.md** (הכוכב הצפוני), ואז המסמך הרלוונטי למשימה.

## הבחנה חשובה
המסמכים ב-🧭 הם **עקרונות קבועים** — אל תשנה אותם כתגובה לשינוי תוכן בודד. המסמך ב-📸
מתאר **מצב נוכחי** ומתעדכן ככל שהקורס מתפתח. אם מצב נוכחי סותר עיקרון — העיקרון מנצח,
או שמעדכנים אותו במפורש ובכוונה.
