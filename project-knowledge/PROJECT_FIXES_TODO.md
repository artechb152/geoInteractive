```text
Work lesson by lesson, in ascending order.
For each issue you fix, change its checkbox from unchecked to checked and add a short implementation note under the item.
If an issue is only partially fixed, keep it unchecked and add a note starting with "Partial:" explaining what remains.
Do not delete completed items.
Keep the file sorted by lesson number.
Preserve the Hebrew wording unless a rewrite is explicitly required.
```
